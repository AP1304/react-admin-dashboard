# React Admin Dashboard

A modern **Admin Dashboard Application** built using **React, TypeScript, Vite, and PrimeReact**.
The project demonstrates scalable frontend architecture with authentication, analytics dashboards, and user management features.

---

## Live Demo

https://react-admin-dashboard-6jg525nsh-amitrp2001-6261s-projects.vercel.app/

---

## Features

• Authentication system with protected routes
• Admin dashboard with analytics widgets
• User management module (search, edit, delete)
• Data visualization with charts
• Reusable component architecture
• Lazy loading for performance optimization
• Dark mode support
